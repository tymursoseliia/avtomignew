# План реализации полноценного сайта под Яндекс Директ и SEO

Этот документ представляет собой подробную спецификацию и пошаговый план разработки. Его можно скопировать и вставить как **главный промпт (инструкцию) для ИИ** в новом проекте, чтобы нейросеть сразу поняла задачу, архитектуру и требования.

## 1. Описание задачи и Технологический стек
**Цель**: Создать максимально премиальный, конверсионный и полнофункциональный сайт для компании по пригону авто из Европы под ключ. Сайт должен быть идеально оптимизирован под рекламный трафик из Яндекс Директ, а также под SEO (органический трафик).
**Дизайн**: Тёмно-синие тона (для вызова доверия), современный UI/UX, стеклянный морфизм (glassmorphism), плавные микро-анимации, полная мобильная адаптация и высокая скорость загрузки.

**Рекомендуемый стек для нового проекта**:
- Фреймворк: **Next.js (App Router)** — идеален для SEO и скорости.
- Стилизация: **Tailwind CSS** (или Vanilla CSS по желанию).
- Логика: **React** (компонентный подход).

## 2. Структура сайта и Якорные блоки (для быстрых ссылок Яндекса)
Для соответствия требованиям Яндекс Директа и создания расширенных объявлений, сайт должен содержать следующие якорные блоки:
- `/#cases` — Каталог авто и Реальные кейсы (с ценами под ключ)
- `/#calculator` — Интерактивный калькулятор стоимости доставки и оформления
- `/#steps` — Как мы работаем (5 этапов подбора, проверки и доставки)
- `/#reviews` — Отзывы (текстовые и обязательно **Видео-отзывы**)
- `/#team` — Наша команда / Менеджеры
- `/#contacts` — Контакты (с виджетами Telegram и WhatsApp)

*Дополнительные страницы*:
- `/thank-you` — Страница "Спасибо" после успешной заявки.
- `/policy` и `/offer` — Юридические страницы.

## 3. SEO и Контент
Главная страница должна иметь строгую семантику:
- **Title**: Пригон авто из Европы под ключ в РФ — расчёт стоимости
- **Description**: Подбор, проверка, доставка, растаможка и оформление авто из Европы. Рассчитайте стоимость автомобиля под ключ и получите 3 варианта под ваш бюджет.
- **H1**: Пригон авто из Европы под ключ в РФ за 14–20 дней
- **H2-блоки**: Рассчитайте стоимость авто из Европы, Реальные кейсы доставленных автомобилей, Как проходит пригон авто под ключ, Почему нам доверяют, Частые вопросы.

*Уточнения для рекламы (отразить в контенте)*: Работаем по договору, Проверка VIN, Фото/видео отчёт, Без скрытых платежей, Доставка в РФ, Подбор под бюджет.

## 4. Интеграция Яндекс Метрики и Цели
В `<head>` сайта должен быть заложен код Яндекс Метрики (с заглушкой ID `XXXXXXX`, если он еще неизвестен). 
Обязательна настройка JavaScript-событий для следующих целей Я.Метрики:
1. `lead_form_submit` — Отправка любой формы связи.
2. `quiz_complete` — Завершение квиза.
3. `click_whatsapp` — Клик по кнопке/ссылке WhatsApp.
4. `click_telegram` — Клик по кнопке/ссылке Telegram.
5. `click_phone` — Клик по номеру телефона.
6. `thank_you_page` — Переход на страницу "спасибо".

## 5. Система сбора заявок и UTM-метки
Сайт должен захватывать и сохранять все **UTM-метки** (`utm_source`, `utm_medium`, `utm_campaign`, `utm_content`, `utm_term`) при входе пользователя и проносить их через весь сеанс (используя `localStorage` или `cookies`).

При отправке любой формы или квиза должны собираться следующие данные:
- Имя, Телефон, Удобный мессенджер.
- Желаемая марка авто, Бюджет, Город доставки.
- Комментарий, Страница захвата.
- Все сохраненные UTM-метки.

Данные должны программно отправляться:
1. В Telegram-бота или чат (через Webhook/API).
2. На Email.
3. Опционально: интеграция с Google Sheets или CRM.
*После отправки — редирект на `/thank-you`.*

## 6. Ключевые функциональные модули для разработки
- **Модуль "Каталог авто"**: Карточки автомобилей с фото, характеристиками и ценой "под ключ". Возможность фильтрации.
- **Модуль "Квиз" (Quiz)**: Пошаговая форма подбора автомобиля с прогресс-баром, в конце запрашивающая контакты для получения подборки.
- **Модуль "Калькулятор"**: Интерактивный просчёт стоимости растаможки и доставки в зависимости от цены авто, года выпуска и объема двигателя.
- **Модуль "Медиа-Отзывы"**: Слайдер с видео-отзывами (интеграция плеера RuTube/YouTube) и текстовыми рецензиями.
- **Блок "Команда (Менеджеры)"**: Карточки сотрудников для повышения доверия (фото, имя, должность, кнопка связи с конкретным менеджером).
